## 前言

基于真实项目 **MiMo TTS Studio**（`~/2026/code/tts-tool/`）的 Electron + Vue 3 桌面应用开发实战记录。

项目技术栈：Electron 39 + ee-core（主进程框架）+ Vue 3.5 + Vite 5 + Naive UI + Pinia，集成小米 MiMo TTS 引擎和 VoxCPM 本地模型。

每篇文章基于实际源码，聚焦一个技术模块，Q&A 格式，附带关键代码片段。

## 技术亮点总览

```
MiMo TTS Studio
├── 🏗️ 架构层
│   ├── electron-egg MVC 框架驱动（主进程）
│   ├── Vue 3 Composition API + Pinia + Router（渲染进程）
│   └── contextBridge 安全 IPC 通信桥
├── ⚙️ TTS 引擎层
│   ├── 双 Provider 策略模式（MiMo 云端 + VoxCPM 本地）
│   ├── LRU 缓存 + 指数退避重试
│   └── 4 种音色设计模式（标准/参数/克隆/导演）
├── 🎨 前端交互层
│   ├── Naive UI 暗色主题系统
│   ├── ActivityBar + Sidebar + Workspace 三栏布局
│   ├── 键盘快捷键系统 (useKeyboardShortcuts)
│   └── 深浅主题切换 (useTheme)
├── 🚦 并发控制层
│   ├── 手写信号量池（最大 3 并发）
│   ├── 任务队列顺序保证
│   └── 失败重试机制
├── 📊 可视化层
│   ├── Canvas 波形渲染（Retina 适配）
│   └── 迷你播放器（波形 + 元数据解析）
├── 💾 数据持久化层
│   ├── 角色即文件夹（Git 友好设计）
│   ├── 防抖自动落盘
│   └── 智能编码检测（中文脚本乱码解决）
└── 📦 工程化
    ├── Vite HMR 开发热更新
    ├── Vitest 单元测试
    ├── electron-builder 多平台打包
    └── electron-updater 自动更新
```
